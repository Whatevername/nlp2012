NATURAL LANGUAGE PROCESSING PROJECT 2012

Installation
------------

* Windows 

1. Install [Active Python 2.7.2+] [1]
2. Copy the [Source Code][2] in the zip format and save it in your hard drive 
3. Unzip the folder and run "tagging.py"
